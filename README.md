# Prediction-of-Finger-Flexion-from-ECoG
Prediction of Finger Flexion - 4th Brain-Computer Interface Data Competition

This repository contains author's independent studies/works on building a framework for successfully predicting finger flexion from the Electro Corticography Data presented at the 4th Brain-Computer Interface Data Competition.  

For details on the competition, please refer to: http://www.bbci.de/competition/iv/

For a detailed description of the data and the labels: http://www.bbci.de/competition/iv/desc_4.pdf
